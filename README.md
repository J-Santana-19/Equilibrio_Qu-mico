# Equilibrio Químico · Cuaderno Interactivo

Cuaderno interactivo de estudio sobre equilibrio químico: constantes de equilibrio, principio de Le Chatelier, problemas paso a paso y ejercicios de práctica.

## 🔗 Demo en vivo

Una vez publicado con GitHub Pages, el sitio quedará disponible en:

```
https://TU-USUARIO.github.io/equilibrio-quimico/
```

## 📁 Estructura del proyecto

```
├── index.html        # Estructura y contenido del sitio
├── css/
│   └── styles.css    # Todos los estilos visuales
├── js/
│   └── script.js      # Toda la lógica interactiva
└── README.md
```

## 🚀 Cómo publicarlo con GitHub Pages

1. Sube estos archivos a un repositorio de GitHub (manteniendo las carpetas `css/` y `js/` tal cual).
2. Ve a **Settings → Pages** en el repositorio.
3. En "Source" selecciona la rama `main` y la carpeta `/ (root)`.
4. Guarda y espera 1-2 minutos. El link aparecerá en esa misma página.

## 🛠️ Cómo verlo en tu computadora antes de subirlo

Simplemente abre `index.html` con doble clic en cualquier navegador. No necesita servidor ni instalación.
